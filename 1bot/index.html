What makes 1Bot special?

Freedom

Completely free to use for everyone. No paywalls or restrictions. Not only free of charge, but also free as in freedom - it's open source!

Multipurpose

Wide variety of features. Whether it is poking fun at friends, managing permissions, or getting lyrics - 1Bot is here to help.

Easy to use

Uses Discord's latest Application Command features, including Slash Commands and Context Menus. No more memorising commands.

Use Anywhere

Supports user installation - add it to your account and use it anywhere, including DMs and group chats. (Moderator commands excluded).
